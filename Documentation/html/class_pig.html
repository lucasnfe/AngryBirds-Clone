Detailed Description

Management of Pig Characters of game.

Contains pig's life and dust effect game object, contains methods to kill the pig, and action to take on collision

override void Pig.Die ( )
virtual

Kills the pig. If not on simulation, play audio clip for the pig death, Spawn the points granted on pig's death, and adds the trajectory particle with the dustEffect GameObject. Calls GameWorld method KillPig() to kill this pig on the level, and them calls the Die() method from Character Class.

Reimplemented from Character.

void Pig.OnCollisionEnter2D ( Collision2D  collision)
private

When something collides with the Pig, check if the magnitude of the relative velocity of colliding object greater than pig's life, if it is, invokes Die() method after _timeToDie seconds and plays the hurt animation.
