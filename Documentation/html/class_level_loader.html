Detailed Description

Contains methods to load and save created levels on xml file.

Can load all levels from the GeneratedLevels folder in xml format, also can save the current level in xml format.

static void LevelLoader.SaveXmlLevel ( ABLevel  level)
static

Saves and ABLevel to an xml file. Writes the label Level, and then the label BirdsAmount together with the amount of Birds in level, then writes the label GameOjbects and for each object writes GameObject, its label, its x position and its y position. For the name of the file, it is added one to the size of current levels saved on the GeneratedLevels folder. Then the file is writen.
