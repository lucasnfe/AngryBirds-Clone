Detailed Description

Agent that makes the actions to shoot the bird.

Calculates the shot data using TrajectoryPlanner, adds noise if last shot missed, and make all the actions necessary to shoot the bird to a pig

void BirdAgent.Update ( )
private

At every Update, if going to throw a bird, drags it to drag position. Next step is to add elapsed time since last update to _throwTimer and if greater than drag time for next shot, sets IsThrowingBir to false and launches the _currentBird.
