bool AngryBirdsGen.Equals ( AngryBirdsGen  otherGen)

Compares this genome to another. First comparing birds amount, then comparing amount of total objects. If both are equal, compares the height of each stack, and, if equals, the object at each position of each stack. If everything is equal, they are equal genomes.
